# Web Traversal And Token Reuse Lab

## Summary

A PHP document portal with naive path normalization, filter-bypass traversal, and a second-stage protected export route.

## Recommended Tools

- `ffuf`
- `dirsearch`
- browser
- Burp Suite
- `curl`

## Learning Goals

- identify and test a file download primitive
- bypass naive traversal filtering
- chain file disclosure into header-based protected data access
